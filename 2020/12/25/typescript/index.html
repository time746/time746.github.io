ts介绍

ts是什么:
可以理解为添加了一些新功能的js

ts能干嘛:
支持ECMAscript最新的特性
js能做的它都能做
可以在代码运行前检查代码是否有误

ts的使用:

  1. 安装
    npm install typescript -g
  2. 然后就可以新建一个ts文件编写了

类型基础

ts可以在变量声明的时候就决定变量的类型, 如果之后修改变量的时候, 类型不对会报错

数组

  • 元组tuple
    用来干什么: 定义数组中具体位置, 每个位置的string , 上面的方法是定义整个数组中的所有元素都应该是number

  • 枚举 enum
    作用: 既可以通过索引得到内容, 又可以通过内容得到索引

  • any
    作用: 如果不确定后面这个变量的类型, 可以使用

  • void 表示没有任何类型
    作用: 一般用在函数中, 没有返回值时使用

  • 默认情况下null和undefined是所有类型的子类型。 就是说你可以把 null和undefined赋值给number类型的变量。
