作用域

  1. 什么是作用域: 作用域是代码中定义变量的区域
  2. 作用域是干什么的: 规定了如何查找变量

js是词法作用域, 作用域在函数定义时就已经固定了

执行上下文

当js执行到可执行代码的时候, 就会创建一个执行上下文, 并压入执行上下文栈中, 函数执行完成就弹出栈

用数组表示执行栈, 那么上面代码的执行栈的变化为

Stack = []

  1. 全局代码执行: Stack.push(globalContext) // [globalContext]
  2. fn1执行: Stack.push(fn1Context) // [globalContext, fn1Context]
  3. fn1完成: Stack.pop(fn1Context) // [globalContext]
  4. fn2执行: Stack.push(fn2Context) // [globalContext, fn2Context]
  5. fn2执行完成: Stack.pop(fn2Context) // [globalContext]
  6. 全部代码执行完成 Stack.pop() // []

可执行代码: 全局代码 , 函数代码, eval代码

简单搭建一个node服务器

Posted on 2020-12-22

搭建本地服务器实现api获取数据

前端在做项目的是有有时候因为没有接口而不好做下去, 这里使用node搭建本地服务简单实现

  1. 首先需要安装express 和 express-generator
    npm install express -g
    npm install express-generator -g

  2. 使用express快速生成项目架构
    express "project-name"

  3. 创建好后会提示先进入项目, 然后执行 npm install
    cd project-name
    npm install

  4. 启动项目
    npm start

  5. 然后就可以去浏览器查看了
    输入127.0.0.1:3000
